# Portfolio Boubacar Nanky

Portfolio dynamique développé en PHP, MySQL et PDO.

## Technologies utilisées
- PHP 7.4+
- MySQL / PDO
- Sessions PHP
- HTML5 / CSS3

## Structure du projet
- Partie 1 : Portfolio statique HTML/CSS → branche partie-1
- Partie 2 : Migration vers PHP → branche partie-2
- Partie 3 : Version dynamique avec BDD → branche main

## Installation locale
1. Cloner le dépôt
2. Copier config/connexion.example.php en config/connexion.php
3. Remplir vos identifiants de base de données
4. Importer database.sql dans phpMyAdmin
5. Lancer avec XAMPP
